What is HTML, CSS and the DOM?

HTML is the standardised language used to create web content.

CSS (Cascading Style Sheets) is a style sheet language how HTML elements will be displayed. CSS allows HTML content to be easily restyled in many different ways without having to edit the HTML when an html page is linked to a style sheet.

The DOM (Document Object Model) is either magic or an API. A browser interprets html or xml documents in the DOM, organising elements of the document into nodes in a tree structure. The nodes are objects that can be manipulated, which I think is what gives web content the special sauce of interactivity and supports the more complex progamming function of (for example) Javascript.

What is boxifying design?

boxifying design simply means breaking down a design into a series of boxes in order to design CSS layout/positioning

What is the box model?

Every html element essentially sits inside a box, like a precious package. Inside the box is content, surrounded by a layer of padding, a border and finally, a margin. Each of these elements can be given different properties such as width, height, colour, and the combination of all these can significantly change the way an element looks and the overall design of a page.
