Technical Blog Part 2 (Udacity Reflection)

What is grid based design?

Dividing css layout into rows and columns in a grid layout makes positioning much easier.

What's all the hype about responsive webpages?

A responsive site will display content to fit any screen size from mobile to desktop. A responsive grid design supports this by dividing the page into percentages rather than fixed widths.

What is semantic structure?

Semantic structure is important in any language. English sentences, for example are typically structured subject - verb - object, e.g. Anna -learns - webdev. That sentence is less meaningful (or has a different meaning) if I rearrange the semantic structure. Likewise, HTML has many semantic elements such as h1, h2, em, href that help us to make sense of HTML. Section elements such as head, body, nav structure the document as a whole.
