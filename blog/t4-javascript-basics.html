How does JavaScript compare to HTML and CSS?

While html and css are used to mark up and style web pages, javascript adds interactivity. If html gives a page structure, css controls style and javascript controls behaviour.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow statements enable a program to conditionally execute code. e.g. If hungry - then eat a snack, else if !hungry, dont snack - else if hungry && bored, drink more water.

These loops need to be broken in order to avoid become trapped in an infinite cycle of snacking, starvation, or hydration.

Explain the difference between accessing data from arrays and object literals.

We access elements of an array by referring to the element's index number:

e.g. var name = weekend[Saturday, Sunday];

Saturday is called by the index number 0, Sunday is called by the index number 1.

Object literals have name values, so the object literals for the variable weekend would be called by the strings "Saturday" and "Sunday"
