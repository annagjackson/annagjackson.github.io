JS Language

parentheses ()

Execute a function. e.g. var scramble = mixItUp(). Parentheses also set parameters, e.g. function myFunction(parameter1, parameter2, parameter3). In a function call, these parameters are called arguments

brackets []

Are for arrays such as var animals = ['cats', 'dogs,''mice', 'sheep']

braces {}

Define the scope of a function. e.g.

function scrambledEggs(){

var mix = mix the eggs;

Return breakfast

}

Variables inside the braces are local, but if we define the variable mix before the function the scope is global.

single quotes ' ' vs double quotes""

There's no difference in javascript, but it pays to be consistent. Quotes are used to define strings e.g. console.log("hello world").
